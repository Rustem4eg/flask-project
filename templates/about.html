О проекте

Этот проект создается в рамках изучения веб-разработки на Flask.

Технологии

  • Python 3.x
  • Flask framework
  • SQLite database
  • HTML5/CSS3
  • Git и GitHub

Цели проекта

  1. Изучить основы веб-разработки на Flask
  2. Освоить работу с базами данных
  3. Научиться использовать систему контроля версий
  4. Создать полноценное веб-приложение
